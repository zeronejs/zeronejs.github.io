### 像数 1, 2, 3 一样容易

```bash
    npm i -g @zeronejs/cli
    zerone new project-name
```

打开项目安装依赖并运行

```bash
    cd project-name
    yarn
    yarn start
```

打开浏览器并导航到 [http://localhost:5000/](http://localhost:5000/)
