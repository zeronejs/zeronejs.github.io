# 目录

## 目录结构

你的项目结构可能是这样：

```
├─ public  // 静态资源文件夹
├─ src 
│  ├─ api // 程序api
│  ├─ common // 通用逻辑或工具
│  ├─ app.controller.ts   // 具有单一路由的基本控制器。
│  ├─ app.module.ts       // 应用程序的根模块。
│  ├─ app.service.ts      // 具有单一方法的基本服务。
│  └─ main.ts             // 应用程序的入口文件。
├─ .gitignore
├─ ...
└─ package.json
```
