# 角色授权

授权是指确定用户能够做什么的过程。

例如，允许管理用户创建、编辑和删除帖子。非管理员用户仅被授权阅读帖子。

# 基本的 RBAC 实现

  • 步骤1: 安装Zerone 提供的一个简单的权限验证包
yarn add @zeronejs/role-easy
